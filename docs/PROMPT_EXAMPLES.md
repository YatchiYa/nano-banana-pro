# Veo 3.1 Prompt Examples & Best Practices

## Prompt Structure Template

Use this structure for best results:

```
[SUBJECT] [ACTION] in [SETTING/ENVIRONMENT], [STYLE], [CAMERA MOVEMENT], [LIGHTING/AMBIANCE]
```

## Category: Cinematic Realism

```

- Aspect Ratio: 16:9
- Resolution: 720p

- Negative Prompt: "cartoon, low quality, blurry"

## Prompt Writing Tips

### ✅ DO's

1. **Be Descriptive**
   - Use adjectives and adverbs
   - Paint a clear picture
   - Include specific details

2. **Specify Camera Movement**
   - "tracking shot", "dolly in", "pan across"
   - "aerial view", "POV shot", "close-up"
   - "slow, smooth movement"

3. **Include Lighting Details**
   - "warm golden sunlight"
   - "cool blue tones"
   - "soft studio lighting"
   - "dramatic shadows"

4. **Add Style Keywords**
   - "cinematic", "realistic", "documentary-style"
   - "cartoon", "animated", "3D render"
   - "film noir", "sci-fi", "fantasy"

5. **Use Composition Terms**
   - "wide shot", "close-up", "medium shot"
   - "two-shot", "single-shot"
   - "shallow depth of field"

6. **Include Audio Cues** (for dialogue/sound)
   - Use quotes for dialogue: "What did you find?"
   - Describe sounds: "engine roaring", "rain splashing"
   - Ambient noise: "wind whistling", "birds chirping"

### ❌ DON'Ts

1. **Don't Use Negative Instructions**
   - ❌ "No cartoon style"
   - ✅ Use negative_prompt instead: "cartoon, low quality"

2. **Don't Be Vague**
   - ❌ "A nice video of a cat"
   - ✅ "A fluffy orange tabby cat playing with a red ball in a sunny living room"

3. **Don't Request Impossible Things**
   - ❌ "Multiple camera angles simultaneously"
   - ✅ "Smooth camera transitions between angles"

4. **Don't Overload with Instructions**
   - ❌ "Do this, then do that, then do this"
   - ✅ Use natural narrative flow

5. **Don't Mix Conflicting Styles**
   - ❌ "Photorealistic cartoon"
   - ✅ Choose one primary style

## Negative Prompt Examples

### General Quality
```
low quality, blurry, distorted, pixelated, watermark, text, logo
```

### Style Exclusions
```
cartoon, drawing, sketch, painting, illustration, 3D render
```

### Content Exclusions
```
violence, gore, explicit, offensive, disturbing, dark, scary
```

### Technical Issues
```
glitchy, artifacts, compression, lag, stuttering, slow motion
```

### Combined Example
```
low quality, blurry, cartoon, watermark, distorted, artificial, studio lighting

## Aspect Ratio Guide

### 16:9 (Widescreen)
- **Best for**: Landscapes, cinematic shots, wide scenes
- **Use case**: Desktop viewing, YouTube, cinema
- **Example**: "Tracking drone view of a man driving a red convertible in Palm Springs"

### 9:16 (Portrait)
- **Best for**: Close-ups, character focus, vertical scenes
- **Use case**: Mobile viewing, Instagram Reels, TikTok
- **Example**: "Close-up of a woman's face as she looks out a window with rain"

---

## Duration Guide

### 4 Seconds
- **Best for**: Quick cuts, transitions, short clips
- **Use case**: Social media, ads, highlights
- **Example**: "A quick pan across a beautiful landscape"

### 6 Seconds
- **Best for**: Medium scenes, balanced storytelling
- **Use case**: Instagram Reels, YouTube Shorts
- **Example**: "A person walks through a room and sits down"

### 8 Seconds
- **Best for**: Full scenes, complete narratives, detailed shots
- **Use case**: YouTube, cinema, full storytelling
- **Example**: "A complete scene with dialogue and action"

---

## Resolution Guide

### 720p
- **Best for**: Quick generation, standard quality
- **File size**: ~50-70MB
- **Use case**: Web, social media, previews

### 1080p
- **Best for**: High-quality output, professional use
- **File size**: ~80-120MB
- **Duration**: 8 seconds only
- **Use case**: Cinema, high-end production, archival

## Common Mistakes & Fixes

### Mistake 1: Too Generic
```
❌ "A person walking"
✅ "A confident woman in a red dress walking down a rain-soaked city street at night, neon lights reflecting off wet pavement, cinematic tracking shot"
```

### Mistake 2: Conflicting Elements
```
❌ "A realistic cartoon"
✅ "A 3D animated cartoon style" OR "Photorealistic"
```

### Mistake 3: Unclear Camera Work
```
❌ "Show a car"
✅ "A POV shot from inside a vintage car driving through rain, Canada at night, cinematic"
```

### Mistake 4: Missing Ambiance
```
❌ "A girl with a puppy"
✅ "A close-up of a girl holding an adorable golden retriever puppy in the park, warm sunlight, soft focus background, joyful atmosphere"
```

### Mistake 5: Impossible Requests
```
❌ "A 30-second video"
✅ "An 8-second video" (max duration)
```

## Prompt Refinement Process

1. **Start Simple**
   ```
   "A lion in the savannah"
   ```

2. **Add Details**
   ```
   "A majestic lion walking through tall golden grass in the African savannah"
   ```

3. **Add Style & Atmosphere**
   ```
   "A majestic lion walking through tall golden grass in the African savannah at golden hour, warm sunlight creating dramatic shadows, cinematic camera movement"
   ```

4. **Add Camera & Composition**
   ```
   "A cinematic tracking shot follows a majestic lion walking through tall golden grass in the African savannah at golden hour. Warm sunlight creates dramatic shadows. The camera slowly pulls back to reveal the vast landscape. Realistic, documentary-style cinematography."
   ```

5. **Add Negative Prompt**
   ```
   Negative: "cartoon, low quality, blurry, artificial"
   ```

## Testing & Iteration

### First Generation
- Use simple, clear prompt
- Standard settings (720p, 16:9, 8s)
- Observe output quality

### Refinement
- Adjust prompt based on results
- Try different aspect ratios
- Experiment with resolution

### Optimization
- Use negative prompts to exclude unwanted elements
- Refine camera descriptions
- Add specific style keywords
